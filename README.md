# Chatting Application - (Frontend)

This is a demo Chatting application.

## Requirments

-   NodeJs 14.15+
-   NPM 6.14+ `or` yarn 1.22+

## Technology

-   Next.js 12.1.6+
-   TypeScript 4.8.4+
-   ESLint 8.17.0+
-   Redux 4.2.0+

## How to use

-   Install NodeJS 14+
-   Clone the Repo
-   Run `npm install` or `yarn`
-   Run `yarn dev`
